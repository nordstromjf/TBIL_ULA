  1. Form the matrix \(C'\)

  2. Use polar coordinates to describe the geometric effect of \(C'\text{.}\)

  3. Using the matrix \(P' = \left[\begin{array}{rr} 1 \amp -1 \\ 2 \amp -1 \\ \end{array}\right] \text{,}\) show that \(A = P'C'P'^{-1}\text{.}\)
