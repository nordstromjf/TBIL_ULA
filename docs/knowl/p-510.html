  1. Consider the matrix \(A\) whose reduced row echelon form is given:

    \begin{equation*} A \sim \left[\begin{array}{rrrr} 1 \amp 2 \amp 0 \amp -3 \\ 0 \amp 0 \amp 1 \amp 2 \\ \end{array}\right]\text{.} \end{equation*}

    Give a parametric description of \(\nul(A)\text{.}\)

  2. Notice that the parametric description gives a set of vectors that span \(\nul(A)\text{.}\) Explain why this set of vectors is linearly independent and hence forms a basis. What is the dimension of \(\nul(A)\text{?}\)

  3. For this matrix, \(\nul(A)\) is a subspace of \(\real^p\) for what \(p\text{?}\)
