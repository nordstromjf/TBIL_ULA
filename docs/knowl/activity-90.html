Activity 3.2.5.

  1. Explain why the vectors \(\evec_1,\evec_2,\evec_3\) form a basis for \(\real^3\text{.}\)

  2. If a set of vectors \(\vvec_1,\vvec_2,\ldots,\vvec_n\) forms a basis for \(\real^m\text{,}\) what can you guarantee about the pivot positions of the matrix

    \begin{equation*} \left[\begin{array}{rrrr} \vvec_1 \amp \vvec_2 \amp \ldots \amp \vvec_n \end{array}\right]\text{?} \end{equation*}

  3. If the set of vectors \(\vvec_1,\vvec_2,\ldots,\vvec_n\) is a basis for \(\real^{10}\text{,}\) how many vectors must be in the set?
