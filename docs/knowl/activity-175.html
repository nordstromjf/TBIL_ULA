Activity 4.5.8.

  1. Consider the matrix \(D = \left[\begin{array}{rrr} 0 \amp 0.5 \amp 0 \\ 1 \amp 0.5 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\) and compute several powers of \(D\) below.

    Determine whether \(D\) is a positive matrix.

  2. Find the eigenvalues of \(D\) and then find the steady-state vectors. Is there a unique steady-state vector?

  3. What happens to the Markov chain defined by \(D\) with initial vector \(\xvec_0 =\threevec{1}{0}{0}\text{?}\)

  4. What happens to the Markov chain with initial vector \(\xvec_0=\threevec{0}{0}{1}\text{?}\)
