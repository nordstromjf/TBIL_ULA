Item e.

Consider vectors that have the form \(\vvec + a\wvec\) where \(a\) is any scalar. Sketch a few of these vectors when, say, \(a = -2, -1, 0, 1, \) and \(2\text{.}\) An appropriate geometric description for this set of vectors would be

  1. a line through \(\vvec\) parallel to \(\wvec\text{.}\)

  2. a line through \(\wvec\) parallel to \(\vvec\text{.}\)

  3. a line through \((0,0)\) parallel to \(\vvec+\wvec\text{.}\)
