Objectives 2.

  • I can translate back and forth between a system of linear equations, a vector equation, and the corresponding augmented matrix.
  • I can compute the solution set for a system of linear equations or a vector equation with infinitly many solutions.
  • I can multiply matrices.
  • I can determine if a Euclidean vector can be written as a linear combination of a given set of Euclidean vectors by solving an appropriate vector equation.
  • I can determine if a set of Euclidean vectors spans \(\IR^n\) by solving appropriate vector equations.
  • I can determine if a set of Euclidean vectors is linearly dependent or independent by solving an appropriate vector equation.
  • I can translate back and forth between a linear transformation of Euclidean spaces and its standard matrix, and perform related computations.
